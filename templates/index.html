Инструкция

1

Откройте Яндекс ID

Зайдите на id.yandex.ru и найдите ваш логин. Введите его в строку поиска и нажмите «Найти плейлисты». Формат логина: Sniper322 или Sniper1337@yandex.ru

2

Выберите музыку

Появится список всех ваших открытых плейлистов, включая любимые треки («Мне нравится»). Кликните на тот, который хотите перенести.

3

Дождитесь переноса

SpotySync автоматически найдет треки в Spotify и создаст из них новый плейлист. Прогресс поиска будет виден на экране.

4

Сохраните плейлист

После завершения вы получите ссылку на Spotify. Откройте её и нажмите «Сохранить», чтобы добавить плейлист в свою библиотеку

💡 Примечание: Плейлист должен быть публичным. Приватные плейлисты недоступны через API. Зайдите в Яндекс Музыку и сделайте их открытыми перед переносом.

О проекте

SpotySync — бесплатный сервис для переноса плейлистов из Яндекс.Музыки в Spotify.

Как это работает

  • Загружаем треки из вашего плейлиста через API Яндекс.Музыки
  • Ищем каждый трек в каталоге Spotify (по названию + исполнителю)
  • Создаём публичный плейлист и выдаём вам ссылку
  • Вы сохраняете его в свою библиотеку — готово!

Технологии

Python FastAPI Spotify Web API Yandex Music API SSE Docker

Ограничения

  • ~92% треков обычно находятся в Spotify (зависит от каталога)
  • Поддерживаются только публичные плейлисты
  • Spotify не всегда точно находит ремиксы и редкие издания

Частые вопросы

Плейлист не находится?

Убедитесь, что плейлист публичный. Откройте его в браузере — если он доступен без авторизации, значит он публичный.

Почему не все треки перенеслись?

Некоторые треки есть только в Яндекс.Музыке и отсутствуют в каталоге Spotify. Обычно переносится ~92% треков.

Могу ли я перенести приватный плейлист?

Пока нет. Мы работаем только с публичными плейлистами, доступными через API без авторизации.
